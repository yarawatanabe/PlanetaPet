# Planeta Pet

Site desenvolvido para a disciplina de Fundamentos de Sistemas Web - Fase 1.

## Descrição

O site apresenta um pet shop fictício chamado Planeta Pet, com informações sobre os serviços oferecidos para cães e gatos.

## Funcionalidades

- Home da empresa com breve apresentação - atualização carrossel de imagens com Bootstrap
- Serviços: Creche, Hotel, Banho e Tosa e Vacinação
- Produtos Cães e Gatos
- Sobre 
- Navegação por âncora entre páginas
- Agendamento com CSS e JS
- Acessibilidade (alt imagem).  

## Tecnologias

- HTML
- Git
- GitHub
- CSS/Bootstrap
- JS
